<objective>
Enable `gsd-set-profile` to apply model profile changes to installed Copilot CLI agent files (`.agent.md`) **without requiring re-installation**.

In Copilot CLI, model IDs are baked into `.agent.md` YAML frontmatter at install time — there is no runtime config lookup. Currently `gsd-set-profile` only writes to `config.json`, which is ignored by Copilot subagents. This plan adds:

1. `resolveModelForCopilot(agentType, profile)` — maps abstract profile names to concrete Copilot model IDs
2. `gsd-tools copilot-apply-profile <profile>` — finds all installed `.agent.md` files and updates their `model:` frontmatter field
3. Updated `set-profile.md` workflow — calls the new command after updating `config.json`

After this plan, `/gsd-set-profile budget` will immediately update agent files. No re-install needed.
</objective>

    ```js
    function cmdCopilotApplyProfile(profile, raw) {
      const validProfiles = ['quality', 'balanced', 'budget'];
      if (!validProfiles.includes(profile)) {
        output({ error: `Invalid profile "${profile}". Valid: ${validProfiles.join(', ')}` }, raw);
        process.exit(1);
      }

      const agentsDir = path.join(
        process.env.GH_COPILOT_CONFIG_DIR || path.join(os.homedir(), '.copilot'),
        'agents'
      );

      if (!fs.existsSync(agentsDir)) {
        output({ skipped: true, reason: 'Copilot agents directory not found', agentsDir }, raw);
        return;
      }

      const files = fs.readdirSync(agentsDir).filter(f => f.endsWith('.agent.md'));
      const updated = [];
      const skipped = [];

      for (const file of files) {
        const agentType = file.replace('.agent.md', '');
        const modelId = resolveModelForCopilot(agentType, profile);
        const filePath = path.join(agentsDir, file);
        const content = fs.readFileSync(filePath, 'utf8');

        const newContent = updateAgentMdModel(content, modelId);
        if (newContent !== content) {
          fs.writeFileSync(filePath, newContent, 'utf8');
          updated.push({ file, model: modelId || '(omitted — inherit)' });
        } else {
          skipped.push({ file, reason: 'no change needed' });
        }
      }

      output({ profile, agentsDir, updated, skipped, totalFiles: files.length }, raw);
    }
    ```

    **4. Add `updateAgentMdModel(content, modelId)` helper function** (private, not exported):
    ```js
    function updateAgentMdModel(content, modelId) {
      // Parse frontmatter block (between first two --- delimiters)
      const match = content.match(/^(---\n)([\s\S]*?)(\n---\n?)([\s\S]*)$/);
      if (!match) return content; // not a valid frontmatter file, skip

      const [, openDelim, frontmatter, closeDelim, body] = match;

      // Remove any existing model: line
      let newFrontmatter = frontmatter.replace(/^model:.*\n?/m, '');

      // Insert model: line if we have a concrete ID
      if (modelId) {
        // Insert after user-invocable line if present, else append before end
        if (newFrontmatter.includes('user-invocable:')) {
          newFrontmatter = newFrontmatter.replace(
            /(user-invocable:.*)/,
            `$1\nmodel: ${modelId}`
          );
        } else {
          newFrontmatter = newFrontmatter.trimEnd() + `\nmodel: ${modelId}`;
        }
      }

      // Normalize trailing whitespace to prevent spurious rewrites on repeated calls
      newFrontmatter = newFrontmatter.replace(/\n+$/, '');

      return openDelim + newFrontmatter + closeDelim + body;
    }
    ```

    **Idempotency requirement:** After writing a file, calling `updateAgentMdModel` again with the
